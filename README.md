# Junglegrid Examples

This repository provides a collection of example prompts, configurations, and job templates for working with [Junglegrid](https://junglegrid.ai), an open-source platform for running AI workloads, inference, and batch jobs on GPUs and other hardware. These examples are intended to help developers, researchers, and organizations quickly get started with Junglegrid and accelerate their AI development workflows.

## Contents

- **claude-desktop/** — Claude Desktop configuration examples
- **cursor/** — Cursor IDE configuration examples
- **prompts/** — Example prompts (speckit and custom)
- **inference-jobs/** — Sample inference job configurations
- **batch-jobs/** — Sample batch job configurations

## Quick Start

1. Clone this repository:
	```sh
	git clone https://github.com/junglegrid/junglegrid-examples.git
	cd junglegrid-examples
	```
2. Review the examples in each folder to find templates relevant to your use case.
3. Copy and adapt the configuration files or prompts for your own Junglegrid deployment.
4. For more information, see the [Junglegrid documentation](https://docs.junglegrid.ai).
