# Prompts

Prompt templates for generating valid Jungle Grid jobs and MCP workflows from natural-language requests.

## Files

- [junglegrid-operator-assistant.md](/home/invinc/code/junglegrid-examples/prompts/junglegrid-operator-assistant.md) — turns user intent into a concrete CLI or API request
- [speckit/junglegrid-job-speckit.md](/home/invinc/code/junglegrid-examples/prompts/speckit/junglegrid-job-speckit.md) — a compact spec prompt for producing structured job definitions

These prompts are designed to keep requests workload-first: image, workload type, model size, command, and routing preference rather than exact GPU selection.
