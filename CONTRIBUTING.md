# Contributing to Junglegrid Examples

Thank you for your interest in contributing to Junglegrid Examples! Your contributions help the community share best practices and accelerate AI development with Junglegrid.

## How to Contribute

1. **Fork** the repository and create your branch from `main`.
2. **Add your example** (prompt, configuration, or job) to the appropriate folder.
3. **Document** your example with a README or comments so others can understand how to use it.
4. **Test** your example if possible to ensure it works with the latest Junglegrid release.
5. **Submit a pull request** with a clear description of your changes and their purpose.

## Code Style

- Use clear, descriptive names for files and folders.
- Add comments or documentation for complex configurations or prompts.
- Follow the structure of existing examples when possible.

## Code of Conduct

Please note that this project is released with a [Contributor Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.
