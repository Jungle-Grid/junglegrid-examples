# Cursor

Use Jungle Grid from Cursor through MCP.

## Prerequisites

- Node.js 18+
- Cursor
- A Jungle Grid API key generated in the portal

## Setup

1. Copy [mcp.json.example](/home/invinc/code/junglegrid-examples/cursor/mcp.json.example) into your project-level `.cursor/mcp.json` or Cursor global MCP settings.
2. Replace `jg_...` with a real API key.
3. Restart Cursor or reload MCP integrations.

## Verify

Open the Cursor MCP tools panel and confirm the Jungle Grid server is available.

Reference: <https://junglegrid.dev/docs/mcp>
